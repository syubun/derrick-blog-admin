## 快速使用
1. fork本項目後再clone到本地修改
2. 進入項目目錄執行`yarn`安裝依賴
3. 執行`yarn dev`開始開發
4. 修改完後執行`yarn build`發布到github pages
``` bash
# 安裝依賴包
yarn
# 啟動程序
yarn dev
# 打包上線
yarn build
```
